#Superstore Sales Dashboard (Power BI)

-Project Overview:
This project presents an interactive Sales Dashboard created using Power BI to analyze and visualize sales data from a Superstore dataset. The dashboard helps in understanding key business insights such as sales trends, profit analysis, and customer behavior.

-Objectives:

To analyze overall sales performance

To identify top-performing products and regions

To track profit and loss across different categories

To provide actionable insights for business decision-making

-Tools & Technologies Used:

Power BI – Data visualization and dashboard creation

Microsoft Excel / CSV – Data source

DAX (Data Analysis Expressions) – Calculations and measures

-Dataset Description:

The dataset contains sales-related information such as:

Order Date

Ship Date

Customer Name

Region

Category & Sub-Category

Sales Amount

Profit

Quantity

-Dashboard Features:

📈 Sales Trend Analysis (Monthly/Yearly)

🌍 Region-wise Sales Distribution

🛒 Category & Sub-category Performance

💰 Profit Analysis

🔍 Interactive Filters (Slicers) for dynamic data exploration

-Key Insights:

The West region shows the highest sales performance

Technology category generates the maximum profit

Some sub-categories show loss despite high sales

Seasonal trends observed in monthly sales data

-How to Use

1. Download the .pbix file from this repository


2. Open it using Power BI Desktop


3. Interact with filters and visuals to explore insights

-Conclusion: 
The dashboard provides a clear and interactive way to monitor sales performance and helps businesses make data-driven decisions.
